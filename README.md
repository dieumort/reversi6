reversi6
====
## Overview
solve reversi 6x6

## Requirement
* g++ (>= 4.9.2) --std=c++11
* boost (>= 1.5.0)

## Usage
`./prg e4 c5 b4 e3` :61m30s  
`./prg e4 c5 b4 e3 c2 d2`  :3m3s  
`./prg e4 c5 b4 e3 c2 d2 f3 f4`  :1m20s  
`./prg e4 c5 b4 e3 c2 d2 f3 f4 f5 b3`  :6.3s  
`./prg e4 c5 b4 e3 c2 d2 f3 f4 f5 b3 a3 c1`  1.2s:
